# Which features should I use?

Below is a table on which feature(s) to go for with the target on left. You are still recommended to finish set up [one](set-up-a-form-with-master-image.md) and [two](set-up-a-form-without-master-image.md) before trying out any of them.&#x20;

| Target                                                                                                    | Feature                                                            |
| --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| Forms with a shared format                                                                                | [Form with master image](set-up-a-form-with-master-image.md)       |
| Forms of the same type but different format                                                               | [Form without master image](set-up-a-form-without-master-image.md) |
| Extract key information such as Total Amount, Date, Time, Address, Name                                   | [Auto Extraction Items](../features/auto-extraction-items.md)      |
| <p>Extract data from an area with some post-processing </p><p><strong>(master form required)</strong></p> | [Post-Processing Scripts](../features/post-processing-scripts.md)  |
| Locate a specific text block based on a known string value                                                | [Key Value Pattern](../features/key-value-pattern-detection.md)    |
| Check if a specific text block exists on uploaded documents                                               | [Text Token](../features/token-group.md#text-token)                |
| Check if a logo/icon exists on uploaded documents                                                         | [Image Token](../features/token-group.md#image-token)              |

If some information you are seeking isn't covered by our Auto Extraction Items or you have some other problems to solve, we do provide [customization services](../customization-services.md)!&#x20;
